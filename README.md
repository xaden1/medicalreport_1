## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- Rust & Soroban CLI
- Freighter Wallet Extension

### Local Setup
1. Clone the repository
2. Run `npm install` in the `/frontend` directory
3. Launch via `npm start`
4. Deploy local contracts according to `contracts` documentation (if running local RPC).
